# MiniProject_StudyCafe
스터디카페 관리 시스템 제작 - 스윙 구현

### 스터디카페 시스템 동작 프로세스

**메인화면** : 사용 가능 좌석(0/전체) , 스터디룸(0/전체)

- 로그인창 : 전화번호 - 비밀번호 입력 창, 로그인 버튼
- 전화번호, 비밀번호(숫자4자리) 입력 후 로그인 버튼 누르기 → 로그인 완료, 이용권 선택 화면으로 이동

**이용권 선택 화면**

- 이용권이 존재하지 않는 경우 : 요금제 선택 창
    - **요금제 선택 창** : 존재하는 모든 요금제를 분류별로 표시 → 모든 요금제는 버튼으로 표시
    - 요금제를 선택할 시, 이용권 선택 창으로 이동
- 이용권이 존재하는 경우 : 이용권 선택 창 -보유하고 있는 이용권 보여줌.(버튼 누르면 넘어가게)
- 이용권 선택 완료, 좌석 선택 화면으로 이동

**좌석 선택 화면**

- 좌석 분류 : 사용중 / 사용가능 / 내좌석(미구현)
- 좌석을 보여주면서 사용중, 사용 가능에 대한 표시(색깔로). 내 좌석이 있으면 표시.
- 원하는 좌석을 선택 + 모든 좌석은 버튼으로 표시

**입실/퇴실 관리 화면** : 입실/퇴실 버튼

- 입실 : 입실 버튼 + 본인 전화번호 + 비밀번호 → 입실처리
- 퇴실 : 최실 버튼 + 본인 전화번호 + 비밀번호 → 퇴실처리
